# Go的包导入

## 包和文件夹是一一对应的

1. 一个包对应一个文件夹
2. 一个文件夹只能包含一个包
3. _test.go 除外

## 包名和文件夹名可以不一致,单不推荐

```go
package main

import (
	"fmt"
    //这个是vscode自动添加的, 也可用于解决包名冲突
    //test_pkg 是路径名
	packageName "test_pkg" 
)

func main() {
	fmt.Println("123")
	packageName.PackageFunc()
}

```

## 解决包名冲突

```go

import (
	"math/rand"
	syssort "sort"
	"fmt"
	mysort "xiuye/sort"
)
```

## 编译时使用的是源码还是.a

一个非main包在编译后会生成一个.a文件 -- 可能在临时文件夹也可能在pkg中,取决于你是否使用go install

结论:

1. 第三方包, 使用的是源码. 如果源码不存在,编译报错.
2. 链接的是**临时目录下的.a文件**
3. 标准库包使用的是.a文件,.a的优先级高于.go. 但是源码必须存在.
